Программа предназначена для сортировки слиянием нескольких файлов.
Входные файлы должны содержать целые числа или строки. Данные должны быть записаны в столбик (каждая строка - новый элемент).
Результатом работы программы является новый файл с объединенным содержимым входных файлов, отстортированным по возрастанию или убыванию с помощью сортировки слиянием.
Если содержимое исходных файлов мешает произвести сортировку (например в массиве чисел встречаются строковые значения), эти значения будут исключатся, пользователю будет выдано сообщение. Наличие таких данных не прерывает сортировку.
Программа написана на языке Java, с использованием библиотеки commons-cli для обработки аргументов командной строки.

Файлы для сортировки необходимо поместить в папку с программой (MergeSortApp.jar).
При запуске сортировки для одного файла (как в примере 1), его содержимое будет отсортировано в указанном порядке и записано в выходной файл.

Параметры программы задаются при запуске через аргументы командной строки (в любом порядке):
1. режим сортировки (необязательный, по умолчанию используется сортировка по возрастанию):
    -a (--ascended) - по возрастанию;
    -d (--descended) - по убыванию;
2. тип данных, обязательный:
    -s (--string) - для строк;
    -i (--integer) - для чисел;
3. имя выходного файла, одно, обязательное:
    -out (--outputFile) <file>
4. имена входных файлов, как минимум одно:
    -in (--inputFiles) <file_1> <file_2> <file_n>

Примеры запуска из командной строки Windows:
java -jar C:\MSA\MergeSortApp.jar -i -a -out out.txt -in in1.txt (для целых чисел по возрастанию)
java -jar C:\MSA\MergeSortApp.jar -s -out out.txt -in in1.txt in2.txt in3.txt (для строк по возрастанию)
java -jar C:\MSA\MergeSortApp.jar -d -s -out out.txt -in in1.txt in2.txt (для строк по убыванию)

Вместо "C:\MSA\MergeSortApp.jar" необходимо указать корректный путь к файлу jar.
При запуске кода из IDE убедитесь, что входящие файлы находятся в родительском каталоге для .class файлов.
Например C:\Users\user\IdeaProjects\MergeSortApp\out\production < тут файлы для обработки, C:\Users\user\IdeaProjects\MergeSortApp\out\production\MergeSortApp\ < тут .class файлы.
